Gulp + Webpack сборка (Relax edition)
=====================

**Для работы нужно установить:**

    * gulp-cli
    * node.js
**Как установить сборку:**

    1. git clone ...
    2. npm i
    3. gulp или npm start
---
Готовое решение для использования в своем проекте ES6, Pug, Scss, а также создания SVG\PNG спрайтов.
